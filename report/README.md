# Reporte del trabajo final

En este directorio se encuentra el informe final del trabajo desarrollado para el curso:

**Algoritmos Recientes para Búsqueda de Proximidad en Altas Dimensiones**  
Doctorado en Ciencias de la Computación – Universidad Nacional de San Luis.

El informe describe el diseño experimental, la generación de embeddings de diálogo y la evaluación de métodos de búsqueda aproximada de vecinos (ANN) utilizando FAISS.

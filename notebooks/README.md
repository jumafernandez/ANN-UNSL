# Notebooks del proyecto

Este directorio contiene las notebooks utilizadas durante el desarrollo del trabajo final.

Las notebooks implementan el pipeline experimental completo:

- **notebook_01_carga_datos_y_embeddings.ipynb**  
  Carga del dataset MultiWOZ 2.2 y generación de embeddings de turno utilizando modelos Sentence Transformers.

- **notebook_02_indexacion_y_ann.ipynb**  
  Construcción y evaluación de índices de búsqueda aproximada de vecinos (ANN) con FAISS.

- **notebook_03_estados_latentes_dinamicos.ipynb**  
  Construcción de estados latentes dinámicos de diálogo y análisis comparativo con embeddings estáticos.
